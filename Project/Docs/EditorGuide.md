# エディターの使い方ガイド

本プロジェクトのエディターは、ImGuiをベースとしたマルチウィンドウ構成になっています。

## メイン画面の構成

エディターは複数の「タブ」で構成されており、用途に応じて切り替えることができます。

- **Develop Tab**: ゲームの構築、デバッグ、パラメータ調整を行うメインのタブです。
- **Game Tab**: 実際のゲーム画面のみを表示します。
- **Prefab Tab**: プレハブの編集専用のタブです。

---

## 主要なウィンドウ（Develop Tab）

### 1. Hierarchy (ヒエラルキー)
シーン内に存在するすべてのエンティティを階層構造で表示します。
- **作成**: 左上の `+` ボタンから新しいエンティティを作成できます。
- **選択**: エンティティ名をクリックすると選択状態になり、Inspectorに詳細が表示されます。
- **プレハブ生成**: `Project` ウィンドウから `.prefab` ファイルをドラッグ＆ドロップすることで、プレハブから実体を生成できます。

### 2. Inspector (インスペクター)
選択中のエンティティが持つコンポーネントとそのパラメータを表示・編集します。
- **Transform**: 位置(Position)、回転(Rotation)、スケール(Scale)の調整。
- **Component**: メッシュレンダラー、ライト、スクリプト、コライダーなどの設定。
- **Script**: C#スクリプトの着脱と、`[SerializeField]` 変数の調整。

### 3. Project (プロジェクト)
アセット（モデル、テクスチャ、プレハブ、音声など）を管理するブラウザです。
- フォルダをダブルクリックして中身を確認できます。
- ファイルをHierarchyにドラッグ＆ドロップして使用します。

### 4. Scene View (シーンビュー)
シーンを自由なカメラ視点で確認・操作できます。
- **操作**:
    - 右クリック + WASD: カメラの移動
    - 右クリックドラッグ: カメラの回転
- **ギズモ**: 選択中のエンティティに対して移動・回転・スケールのギズモが表示され、マウスで直接操作可能です。

### 5. Console (コンソール)
プログラムからのログ、警告、エラーメッセージを表示します。デバッグに活用してください。

---

## 基本操作の流れ

1. **エンティティの配置**: Hierarchyで右クリックするか `+` メニューから作成。またはProjectからPrefabをドロップ。
2. **パラメータ調整**: Hierarchyで選択し、Inspectorで数値を変更。
3. **スクリプトの追加**: Inspectorの `Add Component` -> `Script` を選び、クラス名を入力。
4. **シーンの保存**: Hierarchyウィンドウのメニュー（`+` ボタン内など）から `Scene Save` を選択して保存。
